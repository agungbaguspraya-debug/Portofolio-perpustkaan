# Web Perpustakaan PHP

Aplikasi web perpustakaan berbasis PHP dan MySQL untuk mengelola data buku, anggota, serta transaksi peminjaman dan pengembalian.

## Fitur
- CRUD data buku
- Manajemen anggota
- Peminjaman & pengembalian
- Login admin

## Teknologi
- PHP
- MySQL
- HTML, CSS (Bootstrap/Tailwind)

## Cara Menjalankan
1. Import database ke phpMyAdmin
2. Atur koneksi di file config.php
3. Jalankan di localhost (XAMPP/Laragon)
